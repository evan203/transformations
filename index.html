What is a linear map?

A linear map is a function that maps from one vector space to another that preserves both vector addition and scalar multiplication. That is:

  1. f(u + v) = f(u) + f(v)
  2. f(cu) = cf(u)

What is this visualizer doing?

Consider a linear transformation that maps from R^2 to R^2 defined as T(x)=Ax, where A is a nxn matrix of real numbers. Our input to this function, x, is some vector. Then, we take some object that we wish to transform (i.e. a vector starting at the origin or a star) and apply T to every point on the object.
